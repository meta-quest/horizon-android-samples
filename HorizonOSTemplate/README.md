# Horizon OS Template

A minimal 2D Android application template for Horizon OS on Meta Quest devices. This template
provides a basic Compose UI project structure — no Spatial SDK or VR dependencies.

## Getting Started

1. Open this project in Android Studio with the Meta Horizon OS plugin installed
2. Connect a Meta Quest device or use an emulator
3. Build and run the app

## Project Structure

- `app/src/main/java/` - Kotlin source files
- `app/src/main/res/` - Android resources
- `app/src/main/AndroidManifest.xml` - App manifest with Horizon OS configuration
